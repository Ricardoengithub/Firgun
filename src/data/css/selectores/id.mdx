---
Selecciona el elemento con un id en específico.

---
<br/>

**Se utiliza un hashtag antes del nombre del id para seleccionar.**

Tenemos el siguiente código: 
```html
<h3 id="blue">Heading con id</h3>
<h3>Heading sin id</h3>

```
<br/>

Con el siguiente estilo:

```css
#blue {
    color: blue;
}
```
<br/>

<h3 style="color: blue;">Heading con id</h3>
<h3>Heading sin id</h3>

---
<br/>

- **Este selector debería cambiar únicamente un elemento dentro de la página**
