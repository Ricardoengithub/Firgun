## Font Size
Cambia el tamaño del contenido

---
<br/>

<div style={{textAlign: `center`}}> 
    <h3>Tamaños relativos</h3>
    Los tamaños relativos son recomendables ya que cambian dependiendo del tamaño de pantalla.
</div>

<br/>


**em:**

```css
p{
    font-size: 2em;
}
```
<br/>
<p>Cambia el tamaño del contenido.</p>
<p style="font-size: 2em">Cambia el tamaño del contenido.</p>
