---
Genera espacio hacia el interior desde el borde inferior del elemento raíz.

[Browser support](https://caniuse.com/#search=padding-top)

---
<br/>


Definiremos *border* y *margin* para dar más claridad al ejemplo.

```html
<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-top</p>
</div>
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-top</p>
</div>

<br/><br/>


```css
p{
    padding-top: 50px; /* top */
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0; padding-top: 50px">padding-top</p>
</div>
