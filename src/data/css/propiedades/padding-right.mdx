---
Genera espacio hacia el interior desde el borde inferior del elemento raíz.

[Browser support](https://caniuse.com/#search=padding-right)
---



Definiremos *border* y *margin* para dar más claridad al ejemplo.

```html
<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-right</p>
</div>
```


<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-right</p>
</div>




```css
p{
    padding-right: 50px; /* right */
}
```


<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0; padding-right: 50px">padding-right</p>
</div>
