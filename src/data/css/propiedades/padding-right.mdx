## padding-right
Separa el contenido del borde derecho generando espacio.

---
<br/>

```html
<div>
    padding-right
</div>
```
<br/>

<div style="border: 1px solid blueviolet; display: inline-block;">
    padding-right
</div>

<br/><br/>

```css
div{
    padding-right: 50px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding-right: 50px; display: inline-block">
    padding-right
</div>
