## padding-right
Genera espacio hacia el interior desde el borde inferior del elemento raíz.

---
<br/>


Definiremos *border* y *margin* para dar más claridad al ejemplo.

```html
<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px">padding-right</p>
</div>
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px">padding-right</p>
</div>

<br/><br/>


```css
p{
    padding-right: 50px;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px; padding-right: 50px">padding-right</p>
</div>
