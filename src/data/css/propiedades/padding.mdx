## padding
Genera espacio hacia el interior desde los bordes del elemento raíz.

---
<br/>

Definiremos *border* y *margin* para dar más claridad al ejemplo.

```html
<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px">Padding</p>
</div>
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px">Padding</p>
</div>

<br/><br/>

Definir un solo valor lo aplica a todos lados (top, right, bottom, left).

```css
p{
    padding: 50px;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px; padding: 50px;">Padding</p>
</div>

<br/><br/>

Definir dos valores aplica el primero arriba y abajo(top, bottom), y el segundo a la izquierda y derecha(left, right).

```css
p{
    padding: 50px 100px;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px; padding: 50px 100px;">Padding</p>
</div>

<br/><br/>

Definir tres valores aplica el primero arriba, el segundo a la izquierda y derecha(left, right), y el tercero abajo(bottom).

```css
p{
    padding: 50px 100px 0;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px; padding: 50px 100px 0;">Padding</p>
</div>

<br/><br/>

Definir cuatro valores aplica cada uno en el siguiente orden: arriba, derecha, abajo, izquierda.

```css
p{
    padding: 50px 100px 50px 100px;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 10px; padding: 50px 100px 50px 100px;">Padding</p>
</div>

<br/>
