## padding
Separa el contenido de los bordes generando espacio.

---
<br/>

```html
<div>
    Padding
</div>
```
<br/>

<div style="border: 1px solid blueviolet; display: inline-block;">
    Padding
</div>

<br/><br/>

Definir un solo valor lo aplica a todos lados (top, right, bottom, left).

```css
div{
    padding: 50px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding: 50px; display: inline-block">
    Padding
</div>

<br/><br/>

Definir dos valores aplica el primero arriba y abajo(top, bottom), y el segundo a la izquierda y derecha(left, right).

```css
div{
    padding: 50px 100px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding: 50px 100px; display: inline-block">
    Padding
</div>

<br/><br/>

Definir tres valores aplica el primero arriba, el segundo a la izquierda y derecha(left, right), y el tercero abajo(bottom).

```css
div{
    padding: 50px 100px 0;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding: 50px 100px 0; display: inline-block">
    Padding
</div>

<br/><br/>

Definir cuatro valores aplica cada uno en el siguiente orden: arriba, derecha, abajo, izquierda.

```css
div{
    padding: 50px 100px 50px 100px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding: 50px 100px 50px 100px; display: inline-block">
    Padding
</div>
