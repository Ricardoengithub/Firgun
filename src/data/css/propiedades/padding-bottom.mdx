## padding-bottom
Separa el contenido del borde inferior generando espacio.

---
<br/>

```html
<div>
    padding-bottom
</div>
```
<br/>

<div style="border: 1px solid blueviolet; display: inline-block;">
    padding-bottom
</div>

<br/><br/>

```css
div{
    padding-bottom: 50px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding-bottom: 50px; display: inline-block">
    padding-bottom
</div>
