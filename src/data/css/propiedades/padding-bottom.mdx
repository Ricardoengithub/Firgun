## padding-bottom
Genera espacio hacia el interior desde el borde inferior del elemento raíz.

---
<br/>


Definiremos *border* y *margin* para dar más claridad al ejemplo.

```html
<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-bottom</p>
</div>
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0">padding-bottom</p>
</div>

<br/><br/>


```css
p{
    padding-bottom: 50px;
}
```
<br/>

<div style="border: 1px solid black; display: inline-block;">
    <p style="border: 1px solid blue; margin: 0; padding-bottom: 50px">padding-bottom</p>
</div>
