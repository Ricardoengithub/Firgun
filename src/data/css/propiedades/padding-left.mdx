## padding-left
Separa el contenido del borde izquierdo generando espacio.

---
<br/>

```html
<div>
    padding-left
</div>
```
<br/>

<div style="border: 1px solid blueviolet; display: inline-block;">
    padding-left
</div>

<br/><br/>

```css
div{
    padding-left: 50px;
}
```
<br/>

<div style="border: 1px solid blueviolet; padding-left: 50px; display: inline-block">
    padding-left
</div>
