Si declaramos un arreglo de la siguiente forma:

```js
const [arr, setArr] = useState([]) // arr = []
```

Y después intentamos actualizar el arreglo añadiendo otro elemento de la siguiente forma:

```js
setArr(arr.push(newValue)) // Error
```

Nos dirá que **_arr.push_** no es una función, por lo que entonces ¿Cómo podemos modificar nuestro arreglo?



#### Solución

---

```js
setArr(arr => [...arr, newValue])
```
