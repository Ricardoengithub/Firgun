***useState*** es un hook que nos permite establecer un valor por default a una variable así como una función para su actualización y se declara de la siguiente manera: 

```js
const [item, setItem] = useState(defaultValue)
```

Donde *item* sería el nomble de la variable, *setItem* el nombre de la funcíon para actualizar el valor y *defaultValue* el valor por default de la variable

Su equivalente sería el *this.state* y el *this.setState*
