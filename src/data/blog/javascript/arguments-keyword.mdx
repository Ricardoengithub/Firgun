Nos regresa un arreglo con los valores que le pasamos a una función:

Por ejemplo, en el siguient código:

```javascript
function f(a, b, c, d) {
    console.log(arguments)    /* [Arguments] { '0': 'a', '1': 'b', '2': 'c', '3': 'd' } */ 
    console.log(arguments[0]) /* `a` */ 
    console.log(arguments[1]) /* `b` */ 
    console.log(arguments[2]) /* `c` */ 
    console.log(arguments[3]) /* `d` */ 
    console.log(arguments[4]) /* undefined */ 
}

f('a', 'b', 'c', 'd')
```


Al pasarle las letras 'a', 'b', 'c', 'd', ***arguments*** le asigna un índice a cada parámetro por la posición en que aparece en firma de la función.
