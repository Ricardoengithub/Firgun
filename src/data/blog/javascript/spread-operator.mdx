Tenemos el siguiente código:

```javascript
const f = (a, b, c, d, ...e) => {
    return [a+b, c+d, e]
}

console.log(f(1,2,3,4,5,6,7))  // [3, 7, [5,6,7]] 
```


**Se escribe el nombre de la variable precedida por tres puntos.**

El spread operator nos ayuda a hacer referencia a todos los parámetros que recibimos a partir de cierto punto, en este caso está tomando todos los valores después de *d* y guardandolos en la variable *e*.


```javascript
const x = [1, 2, 3, 4]
const y = [5, 6, 7, 8]
const w = [x, y]
const z = [...x, ...y]
console.log(w) // [[1, 2, 3, 4], [5, 6, 7, 8]]
console.log(z) // [1, 2, 3, 4, 5, 6, 7, 8]
```


No se recomienda usar para concatenar elementos ya que es menos eficiente que concat().

Este operador es útil cuando queremos aplicar los elementos de un arreglo como parámetros de una función, sin este operador tendríamos que hacer lo siguiente:

```javascript
function add(a, b, c) {
    return a + b + c
}
var arr = [10, 20, 70]
add.apply(null, arr)
```


Ya con el operador se simplifica de la siguiente manera:

```javascript
function add(a, b, c) {
    return a + b + c
}
var arr = [10, 20, 70]
add(...arr)
```
