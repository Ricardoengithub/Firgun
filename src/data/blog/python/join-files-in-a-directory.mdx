# Junta todos los archivos en un directorio en uno solo

---
<br/>

El siguiente código resuelve este problema: 

```python{numberLines: true}
from os import listdir   
from os.path import isfile, join

path = "./haskell/"
name = "everything.hs"
onlyfiles = [f for f in listdir(path) if isfile(join(path, f))]

lista = []
for i in onlyfiles:
    f = open(path + str(i), "r")
    lista.append(f.read())
    lista.append("\n \n")
    f.close()

everything = open(path + name, "w")
for i in lista:
    everything.write(i)
everything.close()
```

#### Explicación línea por línea
---

**1:** Añade *listdir* para obtener los archivos en un directorio.

**2:** Añade *isfile* para poder descartar directorios dentro del directorio.

**2:** Añade *join* para concatenar los path de forma inteligente.

**4:** Define el directorio donde están los archivos.

**5:** El nombre del archivo donde se guardarán todos los archivos del directorio.

**6:** Guarda en una lista todos los nombres de los archivos.

**8:** Crea una variable "lista" para los datos de todos los archivos.

**9-13:** Abre cada archivo y lo lee, el contenido se añade a la variable "lista"

**15:** Crea o abre el archivo donde se va a guardar la información.

**16-18:** Escribre los datos de "lista" en el archivo final.
