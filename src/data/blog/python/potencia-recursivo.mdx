El siguiente código resuelve este problema:

```python{numberLines: true}
def potencia(x,y):
	if (y == 0):
		return 1
	elif (y == 1):
		return x
	else:
		return x * potencia(x,y-1)
```



#### Explicación línea por línea

---

**1:** Crea la función potencia que recibe dos numeros(base y exponente):

**2-3:** Caso base para exponente = 0

**4-5:** Caso base para exponente = 1

**6-7:** Caso recursivo dónde se vuelve a llamar a la función.



#### Código completo

---

```python{numberLines: true}
def potencia(x,y):
	if (y == 0):
		return 1
	elif (y == 1):
		return x
	else:
		return x * potencia(x,y-1)

def main():

	x = int(input("Escribe x: "))
	y = int(input("Escribe la potencia a la que quieres elevar x: "))

	print(potencia(x,y)))


if __name__ == "__main__":
    main()
```
