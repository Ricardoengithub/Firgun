El siguiente código resuelve este problema: 

```python{numberLines: true}
def factorial(n):
    fact = 1
    for i in range(2,n+1):
        fact*= i
    return fact
```

#### Código completo
---

```python{numberLines: true}
def factorial(n):
    fact = 1
    for i in range(2,n+1):
        fact*= i
    return fact
    

def main():
    
    n = int(input("Enter a number: "))
    
    if n >= 0:
        print(f"Factorial: {factorial(n)}")
    else:
        print(f"Choose another number")
  

if __name__ == "__main__":
    main()
```
