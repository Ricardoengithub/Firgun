## Color
#### Cambia el color del párrafo

---
##### Ejemplo 1:

```css
p{
    color: purple;
}
```
<br/>


**Sin CSS**

<p>CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>

**Con CSS**

<p style="color: purple">CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>
