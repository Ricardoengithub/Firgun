## Transform
#### Transforma el párrafo

---
##### Ejemplo 1:

```css
p{
    transform: scale(1.5);
}
```
<br/>


**Sin CSS**

<p>CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>

**Con CSS**

<p style="transform: scale(1.5)">CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>
