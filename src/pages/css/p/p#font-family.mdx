## Font family
#### Cambia el diseño de letra del párrafo

---
##### Ejemplo 1:

```css
p{
    font-family: serif;
}
```
<br/>


**Sin CSS**

<p>CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>

**Con CSS**

<p style="font-family: serif">CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</p>
